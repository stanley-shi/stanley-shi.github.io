Today I did some investigation on the Atom Feeds.

Atom Feeds is a alternative to the RSS feed that has been widely adopted around the Internet. It is defined by the IETF.

First of all, it includes two parts: the Atom syndication format and the Atom publishing protocol.

From the format part, you can refer to IETF web site to the detail format of this. One thing I think need users to pay attention is that it support MIME resources as the content.

From the protocol part, the Atom Feed use http as the underlying transport. Using the standard Get, Post, Put, Delete to operate on the resources.

 

Resource link:

Atom standard on WIKI: http://en.wikipedia.org/wiki/Atom_%28standard%29

The Atom syndication format: http://tools.ietf.org/html/rfc4287

The Atom publishing protocol: http://bitworking.org/projects/atom/rfc5023.html
